## 🚀 快速开始

### 1. 启动服务器

```bash
cd /home/terrence/Desktop/battle-survivors
npm install  # 首次运行需要安装依赖
npm start
```

### 2. 打开多个浏览器窗口/标签页

访问：`http://localhost:2567`

打开2-4个窗口/标签页进行测试。

## 🔧 技术细节

### 前端网络管理器 (NetworkManager)

位置：`client/js/utils/network.js`

主要功能：
- WebSocket连接管理
- 自动重连（最多5次）
- 消息队列（断线时缓存）
- 心跳检测（每3秒）
- 玩家同步（平滑插值）

### 后端服务器 (server.js)

位置：`server/server.js`

主要功能：
- Express HTTP服务器
- WebSocket服务器
- 房间管理（2-4人）
- 玩家状态同步
- 游戏逻辑集成

## 🐛 常见问题

### Q: 连接失败？
A: 确保服务器已启动，检查端口2567是否被占用。

### Q: 看不到其他玩家？
A: 确保两个窗口都已连接到服务器，检查浏览器控制台的日志。

### Q: 移动延迟很大？
A: 这是正常的，网络延迟会影响同步速度。可以查看浏览器控制台的延迟显示。
